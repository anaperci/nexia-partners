## O que é este sistema?

O **NexIA Partners** é o sistema onde você registra e acompanha as oportunidades de negócio trazidas por parceiros comerciais. Ele mostra em tempo real quais oportunidades estão ativas, quais estão prestes a vencer e quais já expiraram — tudo de forma automática, sem precisar atualizar manualmente.

O sistema é destinado à equipe comercial, gestores de parcerias e diretoria.

## Como usar — Passo a Passo

### Como registrar uma nova oportunidade

1. No menu lateral, clique em **Oportunidades**
2. Clique no botão **Nova Oportunidade** (azul, no canto superior direito)
3. Preencha o formulário com os dados da oportunidade
4. Clique em **Criar Oportunidade**

| Campo | Obrigatório | Descrição |
|---|---|---|
| Parceiro | ✅ | Nome do parceiro que trouxe a oportunidade. Comece a digitar para ver sugestões |
| Título da Oportunidade | ✅ | Nome ou título da oportunidade (ex: "Licitação TI - Prefeitura") |
| Órgão/Empresa Cliente | ✅ | O órgão público ou empresa que é o cliente final |
| Registrado por | ✅ | Seu nome (já vem preenchido automaticamente) |
| Data de Registro | ✅ | Data em que a oportunidade foi registrada (padrão: hoje) |
| Data de Validade | ✅ | Até quando essa oportunidade é válida |
| Solução Específica | ❌ | Qual produto ou serviço específico se aplica (opcional) |
| Descrição | ❌ | Detalhes adicionais sobre a oportunidade (opcional) |
| Observações | ❌ | Notas internas (opcional) |

> A data de validade deve ser igual ou posterior à data de registro.

---

### Como buscar e filtrar oportunidades

1. Na página **Oportunidades**, use a **barra de busca** para procurar por título, parceiro, órgão ou nome de quem registrou
2. Use o filtro **Todos os status** para ver apenas oportunidades ativas, vencendo ou expiradas
3. Use o filtro **Todos os parceiros** para ver apenas as de um parceiro específico
4. Clique no **nome da coluna** na tabela para ordenar os resultados

---

### Como editar uma oportunidade

1. Na listagem de **Oportunidades**, clique no ícone de **lápis** na linha desejada
2. O formulário será aberto com os dados atuais preenchidos
3. Faça as alterações necessárias
4. Clique em **Salvar Alterações**

---

### Como excluir uma oportunidade

1. Na listagem ou na página de detalhe, clique no ícone/botão **vermelho de lixeira**
2. Uma janela de confirmação será exibida
3. Clique em **Excluir** para confirmar

> Atenção: essa ação não pode ser desfeita.

---

### Como cadastrar um parceiro

1. No menu lateral, clique em **Parceiros**
2. Clique no botão **Novo Parceiro**
3. Preencha o nome (obrigatório) e os demais dados opcionais (email, telefone, empresa)
4. Clique em **Cadastrar**

O parceiro ficará disponível para seleção ao registrar novas oportunidades.

---

### Como exportar dados

1. No menu lateral, clique em **Relatórios**
2. Clique em **Exportar CSV** para baixar uma planilha com todas as oportunidades
3. Clique em **Imprimir / PDF** para gerar um documento imprimível

> O arquivo CSV pode ser aberto no Excel, Google Sheets ou qualquer editor de planilhas.

## Status das Oportunidades

O status de cada oportunidade é calculado **automaticamente** pelo sistema com base na data de validade. Você não precisa atualizar manualmente.

| Status | Cor | Significado |
|---|---|---|
| **Ativo** | 🟢 Verde | A data de validade está a mais de 15 dias de distância |
| **Vencendo** | 🟡 Amarelo | Faltam 15 dias ou menos para a data de validade |
| **Expirado** | 🔴 Vermelho | A data de validade já passou |

> O status muda sozinho conforme o tempo passa. Uma oportunidade ativa hoje pode aparecer como "vencendo" daqui a algumas semanas, automaticamente.

## Dúvidas Frequentes

**Como altero o status de uma oportunidade?**
Você não precisa. O status é calculado automaticamente pelo sistema com base na data de validade. Para mudar o status, edite a data de validade da oportunidade.

**Posso cadastrar uma oportunidade sem vincular a um parceiro cadastrado?**
Sim. Você pode digitar o nome de qualquer parceiro no campo. Se ele já estiver cadastrado, aparecerá como sugestão. Se não, o nome será salvo como texto.

**O que acontece se eu excluir um parceiro que tem oportunidades?**
As oportunidades não são excluídas. Elas continuam no sistema com o nome do parceiro salvo, mas perdem o vínculo direto.

**Posso exportar apenas as oportunidades filtradas?**
O CSV exporta todas as oportunidades do sistema. Para filtrar, você pode usar o Excel ou Google Sheets após o download.

**Esqueci minha senha. Como recupero?**
Entre em contato com o administrador do sistema ou com o time NexIA Lab para redefinir sua senha.

**O sistema funciona no celular?**
O sistema é otimizado para desktop e tablet. No celular, funciona mas a experiência é melhor em telas maiores.
